## Technical Learnings
- Squats at 120s rest is a real performance limiter — research supports 3–5 min for 6–10 rep heavy compounds; bumped to 180s

- Nordic Curl as a mid-workout finisher (exercise 4/5) is wrong context for most lifters
- Duplicate `ex_id` for lateral raises (`lateral_d1` / `db_lateral`) silently splits PR history in the DB tracker — unified to one ID

## Decisions Made
- Program restructure: 3 upper + 1 lower per week (instead of 2+2) — Patrick's preference
- Rest overrides persist in localStorage per `ex_id` — simple, no backend needed
- Existing `customProgram` in localStorage will override new `program.js` defaults until user resets in the program editor
